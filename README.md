# One Year Challenge

Tracking my own product-building challenge over 1 year, in which, I must finish at least 1 MVP product each month.
Conditions:
  1. The source code repository must be public.
  2. The product can be about anything, a library, a MVP app, a website...
  3. A write up post or detailed README must be provided for each product.
  4. ...
